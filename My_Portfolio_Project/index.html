About This Project

This dashboard visualizes customer behavior using RFM Analysis (Recency, Frequency, Monetary value). I processed raw transaction data using Python to segment users into actionable cohorts.

Methodology

  • Step 1: Data Cleaning (Removed returns & null IDs).
  • Step 2: RFM Calculation (Grouped by CustomerID).
  • Step 3: Scoring (Quartile ranking 1-4).
  • Step 4: Segmentation (Logic-based grouping).

Strategic Recommendations

For "Needs Attention": These customers buy often but haven't visited recently. Send them a "We Miss You" coupon immediately.

For "Champions": Reward them with early access to new products to maintain loyalty.
